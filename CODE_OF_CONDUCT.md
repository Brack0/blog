# Code of Conduct

## Our pledge

This is a small, personal, open-source project. In the interest of fostering a welcoming environment, everyone who interacts with this repository — through issues, pull requests, or discussions — is expected to be respectful and constructive.

## Expected behaviour

- Be kind and considerate in all communications.
- Focus on the technical matter at hand.
- Accept feedback gracefully and disagree politely.
- Respect the scope of this project (see [CONTRIBUTING.md](CONTRIBUTING.md)).

## Unacceptable behaviour

- Harassment, insults, or personal attacks of any kind.
- Unsolicited opinions about the author's personal choices, writing style, or content.
- Spam, trolling, or deliberately disruptive behaviour.
- Any form of discrimination based on gender, sexuality, race, religion, disability, nationality, or any other personal characteristic.

## Scope

This Code of Conduct applies within the repository's spaces: issues, pull requests, commit messages, and any other project-related communication.

## Enforcement

Violations may result in comments being removed, contributions being rejected, or the contributor being blocked from the repository — at the sole discretion of the repository owner.

To report a problem, contact the repository owner directly via GitHub.
